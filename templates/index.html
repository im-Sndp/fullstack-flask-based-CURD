[Problem statement]

Develop a pseudo full-stack application in any tech stack which lists and show all words added to the database.

[Solution]

This application is build in respect to the above mentioned problem.The name given to this application is DETool , which basically means data entry tool . This application is capable of adding , deleting and modifying data present in a database.

Technologies used in the application

  • HTML
  • CSS
  • bootstrap
  • Python-Flask

Features inculed in the application

  • Add : To add a word to the database
  • Delte : To delete a word from the database
  • Update : To replace a existing word from the database
  • View : To view all the records in the database

Cases that have been taken in consideration while developing the application

Case 1 : If you try to add more than one word it will give you an error

Case 2 : If you try to enter already existing word in a database it will give you can error

Case 3 : If you try to update a word with already existing word it will give you an error

Case 4 : If you try t0 update one word with multiple it will give you an error
